// Features

Everything you need
for YouTube transcripts

A complete toolkit built right into YouTube's interface. No external apps, no accounts, no data leaving your browser.

Fuzzy Search Across Everything

Powered by Fuse.js — search across titles, channels, tags, and full transcript content with intelligent typo tolerance. Find any moment in any video instantly.

fuse.js

One-Click Extraction

Grab any video's full transcript with or without timestamps in a single click.

AI Translation

Translate into 10+ languages via OpenRouter. Supports Gemini, Claude, GPT, and more.

openrouter

AI Summarization

Get instant summaries of any video. Key points, main topics, and conclusions — all with one click.

AI-powered

Subtitle Overlay

Display translated subtitles directly on the YouTube player in real-time as you watch.

Transcript Library

All transcripts stored locally in IndexedDB. Favorites, tags, and full-text search. No cloud, no signup.

IndexedDB

Dashboard

Full management UI with history, favorites, translations, statistics, and data export/import.

Cost Tracking

See token usage and cost per translation. Know exactly what you're spending via OpenRouter's generation API.

// Get Started

Up and running in 3 steps

No build tools, no configuration. Install and start using it immediately.

1

Install Tampermonkey

Get the Tampermonkey extension for Chrome, Firefox, Edge, or Safari.

2

Add SubScribe

Click the install button above — Tampermonkey will detect and prompt you to install it automatically.

3

Open YouTube

New buttons appear on YouTube's interface. Extract, search, translate — all right there.
